Modelo 02 · Activo

Modelo de mortalidad COVID-19

Módulo demostrativo adaptado para web estática, basado en la lógica de un modelo de clasificación desarrollado en Python. Su objetivo es estimar un nivel de riesgo a partir de variables clínicas, diagnóstico e intervenciones registradas.

Resumen del modelo

Estado Activo
Técnica base Random Forest
Tipo Clasificación
Salida Nivel de riesgo
Enfoque Riesgo

El modelo estima una categoría de riesgo clínico a partir de variables de entrada.

Variables 6

Edad, oxigenación, diagnóstico, síntomas, comorbilidades e intervenciones.

Adaptación Web

La lógica fue convertida a JavaScript para funcionar en GitHub Pages sin servidor.

Variables consideradas

Para esta versión web se utilizan variables representativas del modelo original, transformadas en una lógica interpretativa para permitir la simulación directamente en el navegador.

Nota sobre el modelo

Este módulo es una adaptación académica para una página web estática. No sustituye una valoración médica ni representa un sistema clínico real; su propósito es demostrar cómo un modelo predictivo puede presentarse de manera visual, interpretativa e interactiva.

Modelo 03 · Activo

Predicción de ventas por inversión publicitaria

Modelo de regresión lineal simple que analiza la relación entre inversión publicitaria y ventas. La fórmula se calcula automáticamente desde los datos históricos y se utiliza para estimar ventas esperadas con nuevas inversiones.
